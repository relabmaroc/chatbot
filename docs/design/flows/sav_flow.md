# SAV Flow (Assistance)

1. **Intention** : `sav` ou `reparation`.
2. **Identification** : Nom + Date d'achat (Backend look-up lead/sale DB).
3. **Triage** : 
   - Sous garantie ? -> Procédure de retour gratuite.
   - Hors garantie ? -> Devis de réparation.
4. **Handoff** : Transfert immédiat au technicien si panne complexe.
